JSON Modding Guide

General

As of Mindustry Build v97, modding have been added to the game.

This change made new blocks, turrets, mechs, etc to be possible to add to the game.

You can open mods in your mindustry game by clicking the mods> button, then "import" or Import mod from

There are two types of mods that uses different running methods:

  • JSON Mod
  • Source Mod

This guide will only cover JSON mods, as source mods require a much, MUCH longer tutorial.

JSON Mods

JSON mods are compatible across all platforms as long as the game build is v97 or up

Source Mods

Source mods are compatible across all platforms, but in every version. This modding method can define new things, such as new block behaviors, etc. This modding method is not covered by this modding guide as that would need a lengthy guide.

Mod File

Now, you'll be going to learn how to make a mod file. All you need for this part is a simple text editor. Any one will do, but i suggest Notepad++ for Windows.

First of all, create a file inside your mod folder. It's name must be mod.json, or else your mod won't work.

Then, in the json file, include the following text.

{
"name": "YourModName"
"author": "YourName",
"description": "modDescription",
"version": 1.0
}

This file lets the game identifies this entire folder as a mod. That's why if you name this file other than mod.json, the game will not identify this as a mod, therefore not loading it.
